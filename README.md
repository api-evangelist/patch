# Patch

Carbon removal and offsetting REST API for businesses to purchase high-quality carbon credits, access a marketplace of carbon projects, and integrate sustainability into products.

**Website:** https://www.patch.io/  
**Documentation:** https://docs.patch.io/  
**GitHub:** https://github.com/patch-technology  
**Status:** https://status.patch.io  
**LinkedIn:** https://www.linkedin.com/company/usepatch  
**X:** https://x.com/usepatch  
**Blog:** https://www.patch.io/blog  

## APIs

- **Patch API** — REST API for carbon credit orders, estimates, and project discovery. Base URL: `https://api.patch.io`. Auth: Bearer token.
